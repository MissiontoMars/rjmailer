This directory contains external runtime dependencies of the rjmailer library.
Before each externally downloaded file an SHA1 sum of the file is printed.

The files slf4j-api-1.5.6.jar and slf4j-jdk14-1.5.6.jar are extracted from

a73c602cab0d3c3bd7239de9d89a37eb5b251e03  slf4j-1.5.6.tar.gz

as downloaded from http://slf4j.org/download.html on 2009-04-10. slf4j is
distributed under the MIT licence, a BSD type licence which premits
redistribution.
